# 🍽 Smart Dining System  
## Smart Table Ordering System for Modern Restaurants  

### 🚀 Demo Coming Soon

---

I’m proud to present a fully developed **Smart Table Ordering System** designed for restaurants that want to modernize service and increase operational efficiency.

This is not just a digital menu.

It is a complete **QR-based table ordering and integrated payment platform** that allows guests to order and pay directly from their table.

---

## 🚀 What Guests Can Do

- Browse categorized menu with images and descriptions  
- Order directly from their table  
- Add multiple orders during their visit  
- Send orders instantly to the kitchen  
- View all pending orders  
- Pay by card (Stripe Checkout) or cash  
- Add tips (percentage or custom amount)  
- Respect configurable minimum order rules  
- Use table-based session management  

---

## ⚙ Key Features

- Dynamic menu loaded from database (categories + products)  
- Time-based restrictions (e.g., breakfast until 11:30)  
- Day-based deal validation (weekday offers)  
- Table number session control  
- Multi-order per table logic  
- First order minimum validation (€12 configurable)  
- Card payment minimum validation (€37 configurable)  
- Stripe Checkout integration  
- Tip selection (5%, 10%, 15%, custom %)  
- Mobile-first responsive interface  
- Sidebar category navigation  
- Smart order aggregation before payment  
- Admin panel for full system management
